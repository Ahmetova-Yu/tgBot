# Телеграмм бот для отслеживания местоположения ключа от аудитории в вузе

## Простой интерфейс
- Взаимодействие через интуитивно понятные кнопки
- Минимальное обучение для новых пользователей
- Быстрый доступ ко всем функциям

## Два основных статуса

### "Ключ у меня"
- Отметка о взятии ключа пользователем
- Прозрачное отслеживание ответственности
- Мгновенное уведомление о текущем держателе ключа

### "Ключ сдан"
- Фиксация возврата ключа на место
- Сброс статуса держателя ключа
- Подтверждение доступности ключа для других

## Reply-клавиатура
- Удобное меню прямо в чате Telegram
- Автоматическое отображение после команды `/start`
- Адаптивный дизайн под мобильные устройства
- Быстрый доступ без ввода команд вручную

## Быстрая работа
- Построен на основе **Spring Boot** и **Telegram Bot API**
- Оптимизированная обработка запросов
- Минимальная задержка ответов
- Стабильная работа при высокой нагрузке

# Технологии

## Spring Boot 3.5.7
## TelegramBots Spring Boot Starter 9.2.0
